5/3/1 FOR THE PLATFORM

Wendler's wave-based strength foundation built for strongman competitors. Get brutally strong on the base lifts, then take that strength straight into your events.

  • One wave = 4 weeks — 5s Week → 3s Week → 5/3/1 Week → Deload
  • Each week builds intensity: 65/75/85% → 70/80/90% → 75/85/95%
  • The last set of each week is AMRAP — do more than the minimum if you have it
  • After each wave your Training Max goes up — that's the whole game
  • 4 training days: Log/OHP, Deadlift, Bench, Squat — event work baked in each session
  • Accessory work targets strongman-specific weaknesses: grip, posterior chain, overhead
