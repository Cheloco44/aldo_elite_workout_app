THE 5/3/1 PROGRAM

Jim Wendler's 5/3/1 is built on one principle: get strong by doing the basics, consistently, over time. You train four big lifts — Squat, Deadlift, Bench, OHP — each on their own day, every week, for as many waves as you run.

  • One wave = 4 weeks — 5s Week → 3s Week → 5/3/1 Week → Deload
  • Each week builds intensity: 65/75/85% → 70/80/90% → 75/85/95%
  • The last set of each week is AMRAP — do more than the minimum if you have it
  • After each wave your Training Max goes up — that's the whole game
  • 3 waves (12 weeks) is the recommended starting block
  • Wendler's baseline increase is scaled to your strength level — bigger lifters get bigger jumps. If you want to push harder, the options below add more on top
