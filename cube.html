THE CUBE METHOD

Created by Brandon Lilly, this is the rotation system for powerlifters who want to peak. Each week, you rotate which lift gets the heavy day, explosive day, and rep day. Train hard everywhere — never overtrain anything.

  • Each week: one lift is heavy, one is explosive, one is for reps
  • Heavy day: 85–95% intensity — grind to a top set
  • Explosive day: 50–65% — bar speed is king, no grinding
  • Rep day: 65–75% — high reps, build volume and muscle
  • Runs in 3-week rotations then resets heavier
  • Designed to peak for a competition — most run 10–16 weeks before a meet
